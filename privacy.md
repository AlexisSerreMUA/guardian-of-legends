# Privacy Policy

**Guardian of Legends**
Last updated: June 23, 2026

## Overview
Guardian of Legends is a personal content automation tool that publishes photo carousels to TikTok.

## Data Collection
This application does not collect, store, or share any personal data from TikTok users. The app is used solely by its owner to publish content to a single personal TikTok account.

## TikTok Integration
This app uses the TikTok Content Posting API to publish photo carousel posts. The only data accessed is:
- Basic account information (open_id, display name) required for authentication
- Permission to upload and publish content to the authorized account

## Data Storage
No user data is stored on external servers. Access tokens are stored locally on the owner's device only.
