Fonctionnalités

🤖
Génération IA
Scénarios Top 5 et duels épiques générés automatiquement par Claude AI, basés sur de vraies légendes historiques et mythologiques.
🎨
Visuels Cinématographiques
Images de qualité professionnelle générées par Nano Banana 2 avec texte superposé dynamiquement pour chaque slide.
📱
Publication TikTok
Publication directe sur TikTok via la Content Posting API officielle. Carrousels photo optimisés pour le format TikTok 9:16.
🔐
OAuth Sécurisé
Authentification sécurisée via TikTok Login Kit. Aucune donnée utilisateur stockée sur des serveurs externes.
✅
Validation Humaine
Interface de validation à chaque étape — scénario, images, rendu final — avant toute publication.
📊
Historique & Anti-doublon
Suivi complet des publications. Le système évite automatiquement de répéter des sujets déjà traités.

Comment ça marche

1
Génération du scénario
Claude AI génère un Top 5 ou un duel épique basé sur de vraies légendes mythologiques, historiques ou culturelles. Le contenu est unique et évite les répétitions.
2
Création des visuels
Nano Banana 2 génère des images cinématographiques au format 9:16 pour chaque slide. Le texte est superposé dynamiquement via Sharp.
3
Authentification TikTok
Login Kit OAuth 2.0 permet à l'utilisateur d'autoriser la plateforme à publier sur son compte TikTok de façon sécurisée.
4
Publication automatique
La Content Posting API publie le carrousel directement dans la boîte de réception TikTok du créateur pour révision finale et publication.

Intégration TikTok for Developers

Login Kit Authentification OAuth 2.0 sécurisée pour connecter les comptes TikTok créateurs à la plateforme.
Content Posting API Upload et publication de carrousels photo directement sur les profils TikTok des créateurs autorisés.
user.info.basic Accès aux informations de base du profil (open_id, display name) requis pour l'authentification.
video.publish Autorisation de publication directe de contenu sur le profil TikTok du créateur autorisé.
video.upload Upload de contenu en boîte de réception pour révision et publication manuelle par le créateur.
