How it works

1
Content Generation
The platform automatically generates photo carousels optimized for TikTok.
2
Visual Creation
AI-generated images are created for each slide of the carousel.
3
TikTok Connection
Login Kit allows users to securely connect their TikTok account to the platform.
4
Automatic Publishing
The Content Posting API sends the carousel to the creator's TikTok inbox for final review and publishing.

TikTok for Developers Integration

Login Kit Secure OAuth 2.0 authentication to connect TikTok creator accounts to the platform.
Content Posting API Upload and publish photo carousels directly to authorized TikTok creator profiles.
user.info.basic Access to basic profile information (open_id, display name) required for authentication.
video.publish Permission to directly publish content to the authorized TikTok creator's profile.
video.upload Upload content to the inbox for creator review and manual publishing.
