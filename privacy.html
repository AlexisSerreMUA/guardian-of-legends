Privacy Policy

Last updated: June 23, 2026

Overview

Guardian of Legends is a content creation platform that generates and publishes AI-powered photo carousels to TikTok.

Data Collection

This application does not collect, store, or share any personal data from TikTok users. The app enables creators to publish content to their TikTok accounts.

TikTok Integration

This app uses the TikTok Content Posting API to publish photo carousel posts. The only data accessed is:

  • Basic account information (open_id, display name) required for authentication
  • Permission to upload and publish content to the authorized account

Data Storage

No user data is stored on external servers. Access tokens are stored locally on the user's device only.
